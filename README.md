**FriendChatApp Backend** هو المحرك الأساسي لمنصة دردشة فورية تهدف إلى تعزيز الخصوصية والتواصل السلس بين الأصدقاء. يوفر النظام بيئة آمنة للدردشة الجماعية مع ميزات متقدمة لإدارة الرسائل.

---

## ✨ المميزات الرئيسية (Features)

* **الدردشة الجماعية (Group Messaging):** إنشاء غرف دردشة مخصصة للأصدقاء مع دعم كامل للرسائل الفورية.
* **الرسائل ذاتية التدمير (Self-Destructing Messages):** ميزة برمجية تضمن حذف الرسائل تلقائياً بعد فترة زمنية محددة لتعزيز الأمان.
* **التواصل الفوري (Real-time updates):** استخدام WebSockets لضمان وصول الرسائل وإشعارات الكتابة (Typing indicators) في أجزاء من الثانية.
* **نظام توثيق متقدم (Authentication):** حماية كاملة للمسارات باستخدام JWT (JSON Web Tokens).
* **إدارة الحالة (Online/Offline Status):** تتبع حالة الأصدقاء ومعرفة من هو متاح الآن.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)



- **البيئة التشغيلية:** Node.js
- **إطار العمل:** Express.js
- **قاعدة البيانات:** MongoDB (NoSQL)
- **البرمجة اللحظية:** Socket.io
- **تشفير البيانات:** Bcrypt.js & JWT
